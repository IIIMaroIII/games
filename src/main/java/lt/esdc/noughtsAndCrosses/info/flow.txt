================ Классы ===========================
- Validator
- FileManager
- GameFlow
- Printer
- NoughtsAndCrossesException
================== Поведение ===========================

- init()
    - создать матрицу 3х3
    - заполнить пустыми значениями
    - отпринтить

- startLap(String userMark);
    - попросить у юзера <userMark> ввести ячейку
        - проверить подходит ли под паттерн
            - если нет throw new NoughtsAndCrossesException("Doesn't follow by a pattern")
            - если да -> next
        - public setIndicesPlayerX, в сеттере сделать следующее:
            - перебрать массив индексов и проверить парсится ли значения
                - если нет throw new NoughtsAndCrossesException("The value is not an integer")
                - если да, засетить в state
        - проверить является ли ячейка пустой
            - если нет -> повторить цикл (continue)
            - если да -> next
        - записать <userMark> значение по int[][] indices в матрицу;
        - проверить победу <userMark>
            - если !GameFlow.winner.isBlank()
                - сконфигурировать строку для победителя
                - засетить эту строку в winner
            - если isBlank() -> next
        - проверить tie
            - если да
                - сконфигурировать строку для tie
                - засетить в winner
            - если нет -> next
