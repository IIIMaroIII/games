Цель работы
Создать консольную игру «Крестики-нолики» на Java и сохранять результат
каждой сыгранной партии в файл.

Ограничения

Разрешено использовать:
- условные конструкции (if, else);
- циклы (for, while);
- строки (String);
- обработку ошибок (try-catch);
- работу с файлами (чтение и запись).

Запрещено использовать:
- коллекции (ArrayList, HashMap);
- потоки (Streams);
- графический интерфейс;
- сторонние библиотеки.

Требования к игре

1. Игра должна работать в консоли.
2. Игровое поле — 3×3.
3. В игре участвуют два игрока:
   - первый игрок использует символ X;
   - второй игрок использует символ O.
4. Игроки по очереди вводят координаты клетки.
5. Программа обязана проверять:
   - корректность введённых координат;
   - что выбранная клетка свободна.
6. После каждого хода игровое поле выводится в консоль.
7. Игра завершается в одном из следующих случаев:
   - один из игроков победил;
   - достигнута ничья.

Работа с файлом

8. После завершения каждой игры результат обязательно записывается в файл.
9. В файле должны храниться результаты всех сыгранных игр.
10. Файл не должен очищаться при запуске новой игры —
    данные должны добавляться в конец файла.

Пример записей в файле:
Winner: X
Winner: O
Draw

Повтор игры

11. После окончания игры пользователь должен выбрать:
    - сыграть ещё раз;
    - завершить работу программы.

Обработка ошибок

12. Программа должна корректно обрабатывать:
    - неверный ввод;
    - ввод нечисловых значений;
    - ошибки при работе с файлом.

Результат

Студент должен предоставить:
- программу;
- файл с результатами сыгранных игр;
- демонстрацию работы программы в консоли.
